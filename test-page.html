📱 Integration Code

For your desktop app, use this code to send commands:

// Send commands to the iframe/webview:
const iframe = document.querySelector('iframe');
const iframeWindow = iframe.contentWindow;

// Method 1: Direct function calls
iframeWindow.churchLobbyPlayer.fadeIn(5);
iframeWindow.churchLobbyPlayer.fadeOut(3);
iframeWindow.churchLobbyPlayer.stop();

// Method 2: Custom events
const command = new CustomEvent('churchLobbyCommand', {
  detail: { type: 'fadeIn', seconds: 5 }
});
iframeWindow.document.dispatchEvent(command);
