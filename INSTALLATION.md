# Church Lobby Companion - Installation Instructions

## macOS Installation

### Universal App
This app works on **both Intel and Apple Silicon Macs** - you only need one download!

**Compatible with:**
- Intel Macs (2019 and earlier)  
- Apple Silicon Macs (M1, M2, M3, M4)
- macOS 10.15 (Catalina) or newer

### Security Warning Fix

If you see a security warning when opening the app:

#### Method 1: Right-click to open (Recommended)

1. **Don't double-click the app** - instead, right-click on the Church Lobby Companion app
2. Select "Open" from the context menu
3. Click "Open" when prompted with the security dialog
4. The app will remember this permission for future launches

#### Method 2: System Settings (macOS Ventura 13+)

1. Go to **System Settings** → **Privacy & Security**
2. Scroll down to find the Church Lobby Companion message
3. Click **"Open Anyway"** next to the message
4. Confirm by clicking **"Open"**

#### Method 3: System Preferences (macOS Monterey 12 and earlier)

1. Go to **System Preferences** → **Security & Privacy**
2. Click **"Open Anyway"** next to the Church Lobby Companion message
3. Confirm by entering your password if prompted

#### Method 4: Terminal (Advanced users only)

```bash
xattr -cr "/Applications/Church Lobby Companion.app"
```

### Troubleshooting

#### "App is damaged" error:
1. Download the app again (the file may have been corrupted)
2. Try the terminal command above
3. Make sure you're downloading from the official source

#### "Developer cannot be verified" error:
1. Use Method 1 (right-click → Open) - this is the most reliable
2. If that doesn't work, try Methods 2 or 3 depending on your macOS version

#### App won't open at all:
1. Try restarting your Mac
2. Check if you have enough storage space (app needs ~200MB)
3. Ensure your macOS version is 10.15 (Catalina) or newer
4. Contact support if the issue persists

## Windows Installation

1. Download the .exe file
2. If Windows Defender shows a warning, click "More info" then "Run anyway"
3. Follow the installation wizard

## What the app does

Church Lobby Companion creates a virtual MIDI port that connects to your Church Lobby web app, allowing you to control music playback with hardware MIDI controllers.
